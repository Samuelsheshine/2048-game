# GridMerge - 現代化 2048 益智遊戲

GridMerge 是一款基於經典 2048 玩法開發的現代化 Web 應用程式。本專案結合了流暢的數位合併體驗、直覺的 UI/UX 設計與完整的開發架構。

## ✨ 專案亮點

- **現代化技術棧**：採用 React 18 與 TypeScript 構建，確保程式碼具備強健的型別檢查與可維護性。
- **精美 UI/UX**：使用 Tailwind CSS 與 Shadcn UI 打造響應式介面，並透過 Framer Motion 實現流暢的方塊移動動畫。
- **全方位操控支援**：支援電腦鍵盤方向鍵操作，以及行動裝置的滑動手勢觸控。
- **沉浸式體驗**：內建合併音效與背景音樂，提供玩家更豐富的遊戲回饋。
- **完整後端整合**：使用 Node.js (Express) 搭配 Drizzle ORM，為未來的數據持久化與排行榜功能打下基礎。

## 🛠️ 技術架構

### 前端 (Client)
- **核心框架**: React (經由 Vite 驅動)
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **元件庫**: Shadcn UI & Lucide React
- **狀態管理**: Zustand

### 後端 (Server)
- **環境**: Node.js / Express
- **資料庫管理**: Drizzle ORM
- **開發工具**: tsx (TypeScript 執行環境)

## 🚀 快速開始

### 1. 安裝環境
在專案根目錄下執行以下指令以安裝所有必要的依賴套件：

```bash
npm install
```

### 2. 啟動開發伺服器
執行以下指令同時啟動前端 Vite 與後端 Express 伺服器：

```bash
npm run dev
```
啟動後，請開啟瀏覽器並造訪 `http://localhost:5173`。

### 3. 專案打包
若要建立正式發佈版本，請執行：

```bash
npm run build
```

## 🎮 遊戲玩法
1. 使用 **鍵盤方向鍵** 或在螢幕上 **滑動手勢** 來移動所有方塊。
2. 當兩個 **相同數字** 的方塊碰撞時，它們會合併為一個總和後的方塊。
3. 每一次移動後，版面上會隨機產生一個新的數字方塊（2 或 4）。
4. 拼湊出數字為 **2048** 的方塊即獲得勝利！
